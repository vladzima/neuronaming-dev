How it works?

All names are generated in real time by recurrent neural network, built on scientific computing framework Torch using the LSTM model – mostly for fun and research. 3 million active UK companies data points were used to teach the AI. Industry classification is based on "Standard industrial classification of economic activities (SIC)" and for some industries better results are produced because of a larger amount of companies analysed. Neural network does not copy real world brands thus any such reference is purely coincidental. Contact us.
